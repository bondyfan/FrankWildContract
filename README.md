# Generátor Smlouvy - Frank Wild

Tento projekt je webová aplikace pro generování smluv o spolupráci.

## Jak používat
1. Otevřete `index.html` v prohlížeči.
2. Vyplňte údaje v levém panelu.
3. Klikněte na "ULOŽIT DO PDF" pro tisk nebo uložení.

## Struktura projektu
- `index.html`: Hlavní struktura stránky.
- `style.css`: Styly pro vzhled a tisk.
- `script.js`: Logika pro doplňování údajů.

## Nasazení
Tento projekt je připraven pro nasazení na Netlify. Stačí propojit s GitHub repozitářem nebo nahrát složku.
